만약 컴퓨터에 고딕체가 존재하지 않는다면 Times New Roman이 적용됩니다. 

그리고 font family에는 generic family라는게 존재합니다. 
generic family 는 비슷하게 생긴 글꼴을 모아놓은 그룹입니다.
font family 의 마지막 값은 이러한 generic family를 주는게 좋습니다. 

현재 텍스트의 font-family 는 고딕체, Times New Roman, serif 입니다.
serif는 generic family 이며, 그 안에는 courier, courier new, georgia, times, times new roman 이 포함되어 있습니다. 
따라서 컴퓨터에 고딕체, times new roman 이 존재하지 않아도 serif에 속한 courier 를 적용합니다.

display 속성은 block 과 inline 을 결정합니다. 
기본적으로 block인 요소에게 inline 을 적용하면, 가로 너비를 다 차지하는 block 성질을 잃게된다. 
기본적으로 inline 인 요소에게 block 을 적용해도, block 처럼 다른요소를 포함하는 것은 할 수 없다. 

display 에는 inline, block, inline-block, none 이 있다. 
inline-block 은 inline 이지만 block 처럼 다른 요소를 포함할 수 있다.

position 속성을 통해 요소들의 위치를 정할 수 있습니다. 
position 에는 
static 
relative
fixed 
absolute
4 종류가 있습니다. 
static 은 태그 순서대로 배치하는 방식입니다. 이를 기본위치라고합니다. top, right, left, bottom 을 무시합니다. 
relative 는 기본 위치에서 위치를 옮기는 방식입니다. 상하좌우로 밀어서 옮깁니다. 
fixed 는 viewport 를 기준으로 위치를 설정합니다. background-attachment: fixed 와 동일한 방법입니다. 
absolute 는 ancester 를 기준으로 위치를 설정합니다. ancester 가 없다면 body 태그를 기준으로 설정합니다.
